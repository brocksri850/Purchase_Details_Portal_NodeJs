# MyWorkspace

This is a repository contains API for created using TypeScript 3.2,Express,Node js.

## Install

Install the node packages via:

`$ npm install`

And then run the grunt task to compile the TypeScript:

`$ npm run grunt`

## Starting

To start the server run:

`$ npm start`
